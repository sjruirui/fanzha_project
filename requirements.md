## 项目概述

反诈宣传平台是一个综合性的反电信网络诈骗宣传教育平台，致力于通过多种形式的内容和互动功能，提高全民反诈意识，保护人民财产安全。平台集AI反诈骗助手、AI情景模拟、反诈资讯发布、知识科普、互动交流、在线学习、风险测试、案件举报等功能于一体，为用户提供全方位的反诈服务。

## 技术栈

### 前端技术

- **Vue.js** - 渐进式JavaScript框架
- **Pinia** - 状态管理库
- **Element UI** - 基于Vue的UI组件库
- **Axios** - HTTP客户端
- **Vue Router** - 前端路由管理
- **WangEditor** - 富文本编辑器
- **Highlight.js** - 代码高亮
- **SSE (Server-Sent Events)** - AI助手流式响应通信
- **Vue Lazyload** - 图片懒加载
- **代码分割** - 路由懒加载，按需加载页面组件
- **Gzip压缩** - 减少传输体积
- **CDN加速** - 静态资源分发加速

### 后端技术

- **Node.js** - 后端框架
- **MySQL** - 关系型数据库

- **大模型** - 通义千问AI大模型

## ✨ 项目亮点与创新点

### 1. **多维度反诈内容体系**

- 整合**反诈资讯**、**知识库**、**课堂视频**三大内容模块
- 覆盖骗局曝光、案例剖析、防骗指南、诈骗话术解析等多个维度
- 内容分人群定制（老年人、学生、财务人员等），精准防范

### 2. **寓教于乐的学习方式**

- **反诈课堂**：系统化视频课程，章节式学习
- **反诈自测**：闯关答题模式，边玩边学
- **问答记录**：记录学习进度，巩固知识点

### 3. **互动式社区交流**

- 用户可发布帖子分享防骗经验
- 分类讨论（骗局曝光、骗局避坑、反诈经验交流、受害者求助等）
- 点赞、收藏、评论功能，促进知识传播
- 实时评论回复系统，支持多级评论

### 4. **活动报名与管理系统**

- 支持线上/线下反诈宣传活动发布
- 用户在线报名，系统自动管理
- 活动详情展示（时间、地点、主办方、活动形式等）
- 点赞收藏功能，提升用户参与度

### 5. **案件举报通道**

- 提供便捷的诈骗案件举报入口
- 详细记录举报信息（类型、金额、描述、证据）
- 管理员审核处理机制
- 举报状态跟踪（待处理、处理中、已处理）

### 6. **个性化用户中心**

- 统一管理个人发布、点赞、收藏、评论
- 数据统计展示（发帖数、粉丝数、点赞数等）
- 活动报名记录查询
- 举报历史追踪

### 7. **AI智能助手赋能**

- **AI智能问答**：实时解答反诈问题，提供专业建议
- **AI情景模拟**：模拟真实诈骗场景（冒充公检法、刷单返利、杀猪盘、虚假网贷）
- **流式响应技术**：采用SSE技术，逐字流式输出，体验自然流畅
- **会话管理机制**：智能记录对话历史，保持上下文连贯
- **沉浸式学习**：通过与AI对话互动，在实战中提升防骗能力

### 8. **现代化UI设计**

- 统一的视觉风格，简洁现代
- 响应式布局，适配PC和移动端
- 卡片式设计，信息层次清晰
- 丰富的交互反馈，提升用户体验

## 路由架构设计

### 路由分离策略

本项目采用**路由前缀分离**策略，用户端和管理端共享同一个 Vue 应用，通过路由前缀 `/admin` 区分：

| 端     | 路由前缀 | 说明                       |
| ------ | -------- | -------------------------- |
| 用户端 | `/`      | 面向普通用户的所有功能页面 |
| 管理端 | `/admin` | 面向管理员的后台管理页面   |

### 用户端路由设计

```
/                    → 首页
/news                → 反诈资讯列表
/news/:id            → 资讯详情
/knowledge           → 知识库列表
/knowledge/:id       → 知识详情
/classroom           → 反诈课堂（章节列表）
/classroom/chapter/:id → 章节课时列表
```

## 前端性能优化

### 1. 路由懒加载

- 使用 Vue Router 动态导入，按需加载页面组件
- 减少首屏加载时间，提升用户体验
- 示例：`component: () => import('@/views/News.vue')`

### 2. 图片懒加载

- 使用 Vue Lazyload 插件实现图片懒加载
- 图片进入可视区域时才加载，节省带宽
- 支持加载占位图和加载失败图

### 3. 代码分割

- 将第三方库（Vue、Element UI、Axios等）单独打包
- 业务代码与框架代码分离，便于缓存
- 公共组件提取为独立 chunk

### 4. Gzip压缩

- 服务器端启用 Gzip 压缩
- 减少传输体积，加快页面加载速度
- 压缩 HTML、CSS、JavaScript 文件

### 5. CDN加速

- 静态资源（图片、视频、字体）部署到 CDN
- 减轻服务器压力，提升资源加载速度
- 支持全国多节点分发

### 6. 缓存策略

- 静态资源设置长期缓存（hash文件名）
- API数据合理使用浏览器缓存
- 频繁访问数据使用 Pinia 状态缓存

### 7. 首屏优化

- 关键资源优先加载
- 非关键资源延迟加载
- 骨架屏提升加载体验
